### 1.1.3 返回结果
所有返回结果以 JSON 格式返回；接口返回一共有三种情况：
1. 操作成功后返回，范例：

```json  
{
  "status": "ok",//状态码
  "msg": "SUCCESS"//信息
} 
```  

2. 成功返回数据，范例：

```json  
{
  "data": {//数据
    "name": "minhow",
    "age": "18"
  },
  "status": "ok",//状态码
  "msg": "SUCCESS"//信息
}
``` 
3. 错误返回，范例：

```json  
{
  "status": "ok",//错误状态码
  "msg": "请重新登录！"//错误信息
}
```
